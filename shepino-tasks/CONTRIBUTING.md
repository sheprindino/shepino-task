# Contributing to Shepino Tasks

Thanks for your interest in contributing! 🎉

## How to Contribute

1. **Fork** this repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/shepino-tasks.git`
3. **Make changes** to `index.html`
4. **Test** by opening `index.html` in your browser
5. **Commit**: `git commit -m "feat: your feature description"`
6. **Push**: `git push origin main`
7. **Open a Pull Request**

## Guidelines

- All code lives in a single `index.html` file
- No external dependencies — keep it zero-dependency
- Test on both desktop and mobile browsers
- Follow the existing code style

## Reporting Bugs

Open an issue with:
- What you expected to happen
- What actually happened
- Browser and device info
- Screenshots if applicable
